# Blinkit Sales Dashboard | Power BI Project

## 📌 Project Overview

This project presents an interactive Power BI dashboard developed to analyze Blinkit's grocery sales performance. The dashboard provides insights into sales trends, customer satisfaction, outlet performance, and product distribution, enabling data-driven business decisions through dynamic visualizations and KPIs.

## 🎯 Objectives

* Analyze overall sales performance across outlets.
* Evaluate customer ratings and satisfaction levels.
* Compare sales by outlet size, location, and establishment year.
* Identify top-performing product categories.
* Track inventory distribution and item visibility.

## 📊 Key Performance Indicators (KPIs)

* 💰 Total Sales
* 📈 Average Sales
* 📦 Number of Items Sold
* ⭐ Average Customer Rating

## 🛠️ Tools & Technologies Used

* Power BI Desktop
* Power Query
* DAX (Data Analysis Expressions)
* Microsoft Excel / CSV
* Data Visualization

## 📂 Dataset

The dataset contains grocery sales information including:

* Item Type
* Item Fat Content
* Outlet Location Type
* Outlet Size
* Outlet Establishment Year
* Sales
* Ratings
* Item Visibility

## 📈 Dashboard Features

### 1. Sales Analysis

* Total sales tracking
* Average sales performance
* Sales distribution across outlets

### 2. Outlet Performance

* Sales by outlet size
* Sales by outlet location
* Sales by establishment year

### 3. Product Insights

* Category-wise sales analysis
* Item distribution by product type
* Fat content comparison

### 4. Customer Satisfaction

* Average customer ratings
* Rating trends and performance analysis

## 🔍 Business Insights

* Medium-sized outlets contribute the highest share of sales.
* Tier 3 locations generate significant revenue compared to other regions.
* Fruits, vegetables, and snack foods are among the top-selling categories.
* Customer ratings remain consistently high across most outlets.
* Older outlets tend to generate more stable revenue streams.
